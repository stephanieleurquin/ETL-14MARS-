# Projet ETL Python Simple - Ventes

## Description
Ce projet montre un mini ETL en Python :
1. **Extract** : les données sont créées directement dans un dictionnaire Python.
2. **Transform** : somme des quantités par produit et calcul du rang.
3. **Load / Visualisation** : graphique des ventes par produit avec Seaborn.

## Usage
1. Installer les dépendances pip install pandas matplotlib seaborn
2. Exécuter le script :python scripts/etl_ventes.py
